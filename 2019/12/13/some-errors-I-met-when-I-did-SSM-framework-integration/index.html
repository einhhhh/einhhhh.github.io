0x01写在前头

好久没写博客了，一方面是太忙了，另一方面前面太忙了只是借口主要还是自己太懒了。
最近在学习SSM框架的整合，在配置整个整合框架的过程中，作为初学者的我难免遇到错误，故在此作出记录以在将来作为参考，亦希望能给他人带来帮助。顺便来玩玩安全圈的师傅们为自己的小标题编号时喜欢使用的16进制格式————虽然不是这篇关于安全的文章，嘿嘿。BTW，在这里提一下，打算以后每周在博客总结下自己一周所积累的信息安全知识以及CTF刷题记录，希望自己能做到周更不要太懒。
